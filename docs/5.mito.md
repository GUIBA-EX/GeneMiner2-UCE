# 5. Mito：线粒体基因组

`mito` 使用现有 MainFilter、refilter 与 Rust UCE assembler，从带注释的 GenBank 参考招募 reads，并只在样本 reads 支持时报告闭环。

## 最小命令

```bash
cli/geneminer2 mito \
  -f samples.tsv \
  -o mito_output \
  -p 8 \
  --mito-genbank mitochondrial_reference.gb
```

可选：`--mito-max-reads 320` 限制自适应流程的最大输入；`--no-mito-adaptive-stop` 改为一次性过滤。

## 流程

GenBank gene/rRNA/tRNA + 全基因组/tile bait
→ MainFilter 招募 paired reads
→ 合并为一个 mitochondrial read pool
→ refilter
→ UCE assembler 产生全部 contig
→ overlap、mate-link 与局部 read graph 连接
→ junction read 验证闭环。

参考仅用于招募与 seed；最终序列不会按参考坐标拼接或补洞。因此即使没有近缘参考，只要 bait 能招募到同源 reads，后续组装仍只依赖样本数据。

## 成功标准

成功结果必须同时满足：单一 component、无 `N`、所有连接来自 overlap 或唯一路径、首尾闭合，并有至少 3 条真实 reads 跨越最终 junction。条件不足时保留 partial 输出，但命令返回失败。

## 输出

- `<sample>/mito/mitochondrial_assembly.fasta`：最终闭环或 partial contigs。
- `<sample>/mito/mitochondrial_assembly_summary.tsv`：状态、连接与 junction 支持。
- `<sample>/mito/mitochondrial_mate_links.tsv`：接受的 read-supported links。
- `.gm2_mito_reference/metadata/mitochondrial_genes.tsv`：bait 元数据；坐标为 0-based 半开区间，`segments_0_half_open` 保留跨 origin 的全部片段。

## 专家参数

默认帮助只展示常用参数。bait、overlap、mate-link、bridge 与 junction 阈值仍可作为兼容的专家覆盖项传入；仅在明确诊断招募或闭环失败时调整。完整参数说明见 [旧版详解](mitochondria_ZH.md)。
